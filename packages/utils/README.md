Small shared utility functions used by Kortyx framework packages.

This is a supporting package. Most application code should not need to install it directly.

## Install

```bash
pnpm add @kortyx/utils
```

```bash
npm install @kortyx/utils
```

## Utilities

- `contentToText(...)`
- `deepMergeWithArrayOverwrite(...)`
- `mergeObjects(...)`
- `withRetries(...)`
